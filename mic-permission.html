No prompt? Do it manually
Turn the microphone on manually
If no permission popup appeared, grant it from the extension's site permissions — it takes a few seconds:
  1. Click the puzzle-piece 🧩 Extensions icon in your browser toolbar. This opens the list of your installed extensions.
  2. Click the three-dots ⋮ menu next to WebWright. Or right-click the WebWright icon directly to open the same menu.
  3. Click “View web permissions”. On some browsers this is “Manage extension” → “Site permissions” instead.
  4. Find Microphone in the list and change it to Allow. Leave it set to “Allow” so it stays on for next time.
  5. Restart your browser — close all windows and reopen it. Microsoft Edge and Chrome often don't apply a newly-granted microphone permission until the browser restarts.
  6. Reopen the WebWright side panel and turn Voice Mode on again.

The exact wording varies a little between Chrome, Edge, Brave, Opera, and Vivaldi, but the path is always the same: Extensions icon → ⋮ menu → web permissions → Microphone → Allow → restart browser.

Using Brave? Set up Groq
Set up Groq transcription (free)
Voice transcription needs Groq on Brave — Brave blocks the browser's built-in speech engine for privacy, so even with the mic allowed, the default engine won't hear you. (It's also an option on any browser if you want more accurate transcription.) Setup takes about a minute:
  1. Open console.groq.com and sign in. It's free — you can sign in with a Google or GitHub account.
  2. In the left menu open API Keys → Create API Key, name it, and copy it. The key starts with gsk_. Copy it now — Groq shows it only once.
  3. Back in WebWright, open the side panel and click the Settings ⚙ (gear) icon.
  4. Scroll to the Speech-to-Text (Voice Mode) section.
  5. Set Transcription Engine to Groq Whisper. On Brave this is selected for you automatically once a key is saved.
  6. Paste your key into Groq API Key and click Save.
  7. Turn Voice Mode on (the mic button) and start talking.

Your key is stored locally on your device only. When the Groq engine is active, short audio clips of your speech are sent to Groq for transcription — nothing else.

Which engine should I use?
• Chrome, Edge, Opera, Vivaldi → the default Browser engine works out of the box (no key needed). Switch to Groq only if you want higher accuracy.
• Brave → you must use Groq Whisper (steps above). The built-in engine is disabled there.

Privacy note: the spoken replies (text-to-speech) and everything else stay on your device. Only speech-to-text leaves your machine: the default Browser engine sends audio to your browser's speech service (Google on Chromium), and the Groq engine sends audio to Groq. No audio is stored by WebWright. See the privacy policy for full details.
